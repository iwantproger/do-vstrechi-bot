# Schedule Booking Telegram Bot

Telegram бот для системы бронирования встреч Schedule Booking App.

## Функции

- 📅 Создание расписания встреч
- 🔗 Генерация ссылок для записи
- 📋 Управление заявками
- 🔔 Уведомления о новых бронированиях
- ✅ Подтверждение/отклонение встреч
- 📊 Статистика

## Технологии

- **aiogram 3.x** - асинхронная библиотека для Telegram Bot API
- **aiohttp** - асинхронные HTTP запросы
- **python-dotenv** - управление переменными окружения

## Команды бота

- `/start` - Начать работу с ботом
- `/create` - Создать новое расписание
- `/meetings` - Список встреч
- `/help` - Справка

## Быстрый старт

### Локальная разработка

1. Установи зависимости:
```bash
pip install -r requirements.txt
```

2. Создай файл `.env`:
```bash
cp .env.example .env
```

3. Заполни `.env`:
```
BOT_TOKEN=your-bot-token-from-botfather
BACKEND_API_URL=http://localhost:8000
MINI_APP_URL=http://localhost:3000
```

4. Запусти бота:
```bash
python bot.py
```

## Получение Bot Token

1. Открой Telegram
2. Найди @BotFather
3. Отправь `/newbot`
4. Следуй инструкциям
5. Скопируй полученный токен

## Деплой на Railway

1. Подключи репозиторий к Railway
2. Добавь переменные окружения:
   - `BOT_TOKEN`
   - `BACKEND_API_URL`
   - `MINI_APP_URL`
3. Railway автоматически задеплоит

## Основные обработчики

- `cmd_start` - Приветствие и регистрация
- `cmd_create_schedule` - Создание расписания
- `cmd_meetings` - Список встреч
- `confirm_booking` - Подтверждение встречи
- `decline_booking` - Отклонение встречи
- `inline_query_handler` - Inline режим для шаринга

## Inline режим

Бот поддерживает inline режим для быстрого шаринга ссылок:

1. В любом чате введи: `@your_bot_username`
2. Выбери расписание из списка
3. Отправь сообщение с ссылкой

## Уведомления

Бот отправляет уведомления:
- При новом бронировании
- За 24 часа до встречи (в разработке)
- За 1 час до встречи (в разработке)
- При отмене встречи
